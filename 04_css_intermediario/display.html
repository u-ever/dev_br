CSS DISPLAY:

display: none;
Funciona como se o elemento não existisse. Desaparece como se nunca tivesse sido colocado (diferente do visibility: hidden que só esconde o elemento)

display: block;
Bloco que vai do inicio ao fim da página. Quando o block termina, joga os outros elementos para a outra linha. É possivel alterar o width e o height.

display: inline;
Mantem o pequeno bloco do tamanho do conteúdo, não sendo possível alterar o width ou o height

display: inline-block;
Mantêm o conteúdo na mesma linha porém permite a alteração do width ou do height

NÃO DEVEMOS UM ELEMENTO TIPO BLOCK DENTRO DE UM PARÁGRAFO! Pois causa quebra no fluxo do parágrafo.
