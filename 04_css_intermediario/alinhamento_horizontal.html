Alinhamento Horizontal

Alinhamento de Divs

margin: auto; CENTRALIZAR

margin-left: auto; POSICIONAR À DIREITA

margin-right: auto; POSICIONAR À ESQUERDA (padrão)

Alinhamento de Imagens

Para alterar o alinhamento de imagens com o "margin" é necessário trocar o display para block por exemplo (inline-block não funciona)

Caso seja necessário alterar o alinhamento de um elemento inline é necessário coloca-lo dentro de uma DIV e usar o atributo "text-align"
