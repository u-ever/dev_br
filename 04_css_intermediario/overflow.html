Overflow

Overflow é quando temos um conteúdo dentro de um elemento, porém esse conteúdo é maior que este elemento

overflow: visible; Quando todo o conteúdo é exibido mesmo que saia do elemento

overflow: hidden; Esconde todo o conteúdo que sair estiver saindo do elemento

overflow: scroll; Utiliza um scroll para que todo o conteúdo seja exibido dentro do elemento com a ajuda de um scroll

overflow: auto; Utiliza o scroll apenas onde é necessário no conteúdo

overflow-y ou overflow-x: scroll; utilizado para colocar o scroll no eixo X ou Y
