Alinhamento

Display

display: inline-block; Coloca os blocos na mesma linha (nestes casos podem surgir espaços originiarios da organização do HTML (quando é dado ENTER para que cada elemento fique em uma linha por exemplo.))

Position

A propriedade position altera apenas a posição do elemento que a recebe, não alterando nenhum outro elemento no processo (como ocorre se colocarmos margin, que acaba empurrando os outros elementos) devido a isso, ela pode fazer o elemento ficar em cima de outros elementos.

Static e Relative

position: static; Este é o posicionamento padrão, deixa o block um em baixo do outro e o inline um do lado do outro (grudados)

position: relative; Modifica a posição do elemento em relação a posição "statict" inicial. Porém se a janela mudar de tamanho, a posição será mantida, ou seja: O elemento "filho" não acompanha o elemento "pai".

Absolute

position: absolute; A posição absoluta tira a interação entre o elemento e o restante da página, fazendo com que os outros elementos possam estar por cima do elemento em absoluto

Este posicionamento não mantêm relação com sua posição inicial. Ele mantêm relação direta com o posicionamento do elemento pai, desde de que este NÃO TENHA POSICIONAMENTO STATIC

Caso ele não encontre nenhum elemento acima do elemento pai com posicionamento diferente de static, ele vai se posicionar em relação ao body.

Fixed

postion: fixed; A posição fixed torna o elemento fixo em referência a Window! (vai além do body)

Sticky

position: stiky; (grudento) Esta posição mantem também mantêm o elemento fixo porém ele interage com os outros elementos enquanto eles estiverem visiveis na window (diferente do fixed que faz com que o elemento seja ignorado). Caso os outros elementos sejam ocultos por conta do scroll o elemento sticky continuará visível enquanto os outros saem da window (como se passasem por trás do elemento sticky).
