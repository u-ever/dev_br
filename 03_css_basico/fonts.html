Fontes

Existem 3 grandes familias de tipos de fontes:

  • Com Cerifa
  • Sem Cerifa
  • Monospace - Todos os caractéres com o mesmo tamanho

Para alterar a fonte utilizamos links externos ou arquivos internos. O Google Fonts é um bom exemplo.
Devemos colocar as fontes no style com o atributo font-family (indicando ao menos uma fonte "padrão")
Para fontes externas devemos usar o declarar a fonte baixada no css com o atributo @font-face, dizendo o nome da fonte (que será indicado no font-family) e o local onde o arquivo se encontra.

O site https://www.fontsquirrel.com/ converte fontes para WOFF (padrão mais leve).
